Accomplishments of 2020

A list of what I've achieved in 2020.

It's been a long and rough year. But despite that, I've managed to do something big and great. Much bigger and greater than some of the previous years. So, here we go.

  • One of the things you're looking at right now - the site redesign. Before, it's been pretty rough and dark, with not that much content. But now, I've added a lot of new projects, screenshots and even made some fancy links!
  • Next things on the list are related to the site, because they're hosted on the same platform - GitHub.

    • 1. The biggest projects I'm proud of are, of course, Serious Sam SDK and Custom Entities Library for new Visual Studios. Very nice and modern environment for good old mod making.

    2. Another project which I'm proud of starting this year is a direct successor to my old Advanced Multiplayer mod - Advanced Multiplayer 2! Which is at the time of writing is still in development, with the first public beta already available for download.

    3. Other, not so unimportant projects, are:

    • Dreamy Structures - my own implementation of array, list, map and stack structures on pure C++.
    • Dreamy JSON - a self-made JSON config parser with Dreamy Structures in its base.

    Maybe these projects are not the greatest, but they have been a great learning curve for myself and I find them very useful for my own projects (one of which is Advanced Multiplayer 2).

    4. And finally, a couple of programs that I made as experiments with Windows Forms, a simple minesweeper game and a mouse cursor trapper.

  • Moving forward to projects that aren't on GitHub but still coded by me - Serious Sam mods.

    • This year has probably been the most Serious Sam projects-filled ever. So many projects have been started and even released. You know about Advanced Multiplayer 2 already, how about a few more:

    • Half-Life 2 Immersion Mod - not exactly this year's project, but released in August 2020. An experiment that grew into something amazing that I didn't even expect myself. Not a full version yet but who knows, maybe it will be in 2021.

    • Tiny Mod Pack - this project grew out of several small experimental mods that I've been trying to make. It got frozen when I switched to another project without finishing a couple of mods that would be in this mod pack. Not-so-publicly released on some Discord server, this beta version of Tiny Mod Pack contains 3 out of 5 planned mods: Randomized, Shrunk and Shuffled.

    • Entity Replacement Pack - another Discord-exclusive experiment of trying to make an entity pack that isn't tied to vanilla entities for customizing enemies, items and projectiles. The pack has also been unfinished and abandoned and so it can be pretty buggy. Uses an earlier version of Dreamy JSON for loading model configs. Includes a test map that showcases most of the replacers' features (the grenade launcher uses a custom picked mask value making it unpickable by the second player in multiplayer specifically). If you wish to use it in your own map in this state, use it at your own risk.
