Serious Files

Custom Entities Library for Visual Studio 2013

This is a clean library base for your own Serious Sam entities based on Serious Sam SDK v1.07 by Croteam released in 2002.

This SDK contains only the engine headers and the project files were created for Visual Studio 2013 Professional edition.

Half-Life 2 Immersion Mod

Half-Life 2 Immersion Mod tries to recreate an atmosphere of playing Half-Life 2 on Serious Engine with ported weapons, enemies, effects and a bunch of other features that mimic Half-Life 2 as closely as possible!

Serious Sam SDK v1.07 for Visual Studio 2013

This is the original Serious Sam SDK v1.07 by Croteam released in 2002 that includes the following projects: EntitiesMP, GameGUIMP, GameMP.

The original source code of these projects remains untouched, however the project files were created for Visual Studio 2013 Professional edition and can be compiled using newer Visual Studios.

Advanced Multiplayer Mod

A mod for Classic The Second Encounter that adds a lot of multiplayer features and some kind of a Cooperative support for Singleplayer maps.

Run Or Die: Sensitive Task

Run Or Die: Sensitive Task is a spin-off of a hardcore map franchise by Ar2R-devil-PiNKy - Run Or Die.

The Mental Power of The Universe

An attempt to make a story mod for The Second Encounter as a sequel to Serious Sam 2. It was abandoned back in 2014 but was only released in 2017. The plot is simple, the mod is broken and unfinished and it's also in russian so you probably won't even understand anything.
