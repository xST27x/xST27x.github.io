Run Or Die: Sensitive Task

Release date: 2017-07-24

Run Or Die: Sensitive Task is a spin-off of a hardcore map franchise by Ar2R-devil-PiNKy - Run Or Die.

You wake up in a strange place and suddenly hear some voice in the head. Is it hallucinations? Or maybe a dream? You can't tell. The only thing you can do is to listen to that voice because he, whoever he is, isn't in a mood for jokes. The voice gives you a simple task. And the point of this task is to escape. But will it be as easy as it seems? You'll find out in Run Or Die: Sensitive Task.

The map includes 12 achievements. Can you complete everything? Depends on how thoroughly you explore the map because there's a lot of things that are hidden from the player.
